Upload File

📱 Lưu ý khi upload bằng điện thoại:

- Điện thoại phải kết nối cùng WiFi với server (không dùng 4G/5G).
- Nếu quét QR nhưng không mở được trang → kiểm tra lại IP host.
- Không tắt màn hình khi đang upload file lớn.
- Tránh chuyển app giữa chừng (iOS có thể tạm dừng upload nền).
- Khuyến nghị:
  • File đơn lẻ ≤ 2–3GB
  • Upload tối đa 3–5 file cùng lúc trên mobile
  • WiFi yếu có thể làm giảm tốc độ đáng kể

⚠ Nếu upload file lớn >5GB từ điện thoại, nên dùng máy tính để ổn định hơn.
⚡ Logic Upload hiện tại:
- Upload song song tối đa 5 file cùng lúc (MAX_PARALLEL = 5)
- Các file còn lại sẽ vào hàng đợi (queue)
- Theo dõi tốc độ từng file và tổng tiến trình

📌 Khuyến nghị sử dụng:
- Tối ưu: 5–20 file / lần upload
- File đơn lẻ: nên dưới 5GB
- Tổng batch: nên dưới 50GB để tránh timeout mạng LAN yếu
- Nếu file >10GB: nên upload riêng từng file
⚙ Nếu chạy bằng Docker:
- Phải publish port: docker run -p 8080:8080 image
- Điện thoại và server phải cùng mạng LAN
- Lấy IP host bằng: ip a hoặc hostname -I
- Truy cập: http://IP_HOST:PORT
- Monitor hệ thống: http://IP_HOST/monitor
