QR Code được tạo dựa trên địa chỉ này. Nếu tắt/bật lại mạng, IP có thể thay đổi → cần reload trang để tạo QR mới. Monitor hệ thống:
🐳 Nếu chạy bằng Docker
IP hiển thị phía trên có thể là IP container (vd: 172.x.x.x) và sẽ không truy cập được từ điện thoại.

➜ Lấy IP máy host thật:
Linux: ip a hoặc hostname -I
Windows: ipconfig

Sau đó truy cập dạng:
http://192.168.x.x:8080

📱 Hướng dẫn cho điện thoại - Phải cùng WiFi với server (không dùng 4G/5G).
- Vào Cài đặt WiFi → xem IP của điện thoại.
- Phải cùng tên wifi.
- 3 số đầu phải giống IP server.

Ví dụ hợp lệ:
Server: 192.168.1.5
Phone: 192.168.1.12

⚠ Không tắt màn hình khi upload file lớn. ⚠ iOS có thể tạm dừng upload nếu chuyển app.
⚡ Logic Upload
- Upload song song tối đa 5 file (MAX_PARALLEL = 5).
- File còn lại vào hàng đợi (queue).
- Theo dõi tiến trình từng file và tổng tiến trình.

📌 Khuyến nghị:
- 5–20 file mỗi lần upload.
- File đơn lẻ nên ≤ 5GB.
- Tổng batch ≤ 50GB để tránh timeout LAN yếu.
- File >10GB nên upload riêng từng file.
